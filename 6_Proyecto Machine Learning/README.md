### Proyecto de ML:
# Calificaciones de Estudiantes

Este proyecto está organizado en cuatro notebooks, cada uno cubriendo una etapa del flujo de trabajo de análisis y modelado.

## 1_EDA

- Análisis exploratorioinicial.

- Carga y revisión de datos

- Estadísticas descriptivas

- Visualizaciones básicas

- Detención de valores atípicos y patrones relevantes

- Análisis de inconsistencias

## 2_PreProcesamiento

Preparación de los datos para los modelos.

- Limpieza del dataset

- Gestión de nulos y outliers

- Codificación de variables categóricas

- Escalado/normalización

## 3_Modelo_Regression

Entrenamiento y evaluación del modelo de regresión

- Separación del conjunto de datos

- Entrenamiento del modelo

- Validación del modelo

- Comparación de distribuciones

- Análisis de residuos

- Análisis de la importancia de las características

- Obtención de métricas de evaluación del modelo

- Entrenamiento final del modelo

- Prueba de modelos con regularización

## 4_Modelo_Clasificacion

Entrenamiento y evaluación del modelo de clasificación

- Separación del conjunto de datos

- Entrenamiento del modelo

- Validación del modelo

- Obtención de métricas de evaluación del modelo

- Análisis de la importancia de las características

- Entrenamiento final del modelo

- Prueba de modelos con regularización
